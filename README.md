# mhtp
Allows you to easily download lots of html pages, parse them and save to pdf-file.

#usage 
mhtp.py [-h] [-s] [-p deepness [deepness ...] --] [--no-css]
               [-a] [-q] [-v]
               name url [url ...] [name url [url ...] ...]
               
see description in `mhtp.py --help`

# requirements
 - weasyprint
 - lxml
 - pypdf2
